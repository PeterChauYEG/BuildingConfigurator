## Setup

- install node with nvm
- clone this repo
- run `yarn`

## Managing deps

- Start by running in the love-deployment repo: `yarn deps:up'
- stop by running in the love-deployment repo: `yarn deps:down'

## Running dev

`yarn start:dev`
